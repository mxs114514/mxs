# 前端大作业

写一个记账 App，要求实现下列功能：

1. 记录每笔支出和收入，包括金额、时间、类别、备注等信息
2. 可以查看每日、每月、每年的收支情况
3. 可以查看某一类别的收支情况

推荐技术栈：

1. React
2. Ant Design

## 详细要求

在一个页面上共有三个部分：

1. 一个表单输入区，用于录入支出和收入信息
2. 一个过滤器的区域，用于选择查看的时间范围和类别
3. 一个展示区，用于展示符合条件的支出和收入信息

### 表单输入区

表单输入区包括以下内容：

1. 金额：一个数字输入框，用于输入金额
2. 时间：一个日期选择器，用于选择时间
3. 类别：一个下拉框，用于选择类别
4. 备注：一个文本输入框，用于输入备注
5. 一个提交按钮，用于提交表单

行为：

1. 点击提交按钮后，全部输入框清空，并在展示区显示新的支出或收入信息

### 过滤器区域

过滤器区域包括以下内容：

1. 一个日期范围选择器，共有三个选项：当日、当月、当年
2. 一个类别选择器，用于选择类别

行为：

1. 日期范围选择器默认为当日，选择不同的选项后，展示区域的内容会相应变化
2. 日期范围只能选择一个选项
3. 类型选择默认为 “全部”，选择不同的类别后，展示区域的内容会相应变化
4. 类别只能选择一个选项

### 展示区

展示区包括以下内容：

1. 一个卡片列表或是表格，用于展示符合条件的支出和收入信息
2. 每个卡片或表格行包括：金额、时间、类别、备注

行为：

1. 展示区域的内容会根据过滤器区域的选择变化
2. 展示区域的内容按时间倒序排列
