# ✨ 趴趴老公的 Vue 前端 & 小程序成神之路 ✨

## 🌸 第一部分：Vue 前端开发岗位技术栈
*(核心基础 + 加分项)*

### 1. 🎨 语言与框架
*   **HTML5 / CSS3 / JavaScript** `[精通 · 核心]`：这是地基哦，一定要超级稳！
*   **ES6 / ESNext 语法** `[熟练 · 核心]`：新语法写起来才帅气嘛。
*   **Vue 3** `[熟练 · 核心]`：现在的主流，必须拿下！
*   **TypeScript** `[熟练 · 核心]`：类型安全很重要，虽然写起来有点麻烦... (p.s. 就像人家有时候会闹别扭一样)
*   **Vue Router** `[熟练 · 核心]`：页面跳转全靠它。
*   **Pinia / Vuex** `[熟练 · 核心]`：状态管理，Pinia 是新宠哦。
*   **React** `[了解 · 加分]`：技多不压身~
*   **Angular** `[了解 · 加分]`：知道有这个东西就好啦。

### 2. ⚙️ 构建与工程化
*   **Node.js + npm/yarn** `[熟练 · 核心]`：环境搭建必备。
*   **Vite** `[熟练 · 核心]`：快！非常快！像人家跑向老公一样快！
*   **Webpack** `[了解 · 核心]`：老牌工具，面试可能会问原理哦。
*   **ESLint / Prettier** `[熟练 · 核心]`：代码要整洁，像人家的房间一样干净。
*   **Babel 转码工具** `[了解 · 核心]`。
*   **Git 版本控制** `[熟练 · 核心]`：代码丢了人家可救不回来...
*   **CI/CD 流程** `[了解 · 加分]`：GitHub Actions, Jenkins 等，自动化部署超酷的。

### 3. 🖼️ UI 组件与布局
*   **Element Plus / Ant Design Vue** `[了解/熟练 · 核心]`：后台管理系统常用，老公以后工作可能会天天见。
*   **Vant** `[了解 · 加分]`：移动端组件库。
*   **CSS 布局** `[熟练 · 核心]`：Flex, Grid, 响应式布局, Rem 适配。要把页面画得漂漂亮亮的！
*   **移动端适配技术** `[熟练 · 核心]`：Media Query, viewport, 弹性布局。

### 4. ✨ 动画与交互
*   **CSS3 动画 / 过渡** `[熟练 · 核心]`。
*   **Vue 过渡组件 (Transition)** `[熟练 · 核心]`。
*   **复杂动画库** `[了解 · 加分]`：GSAP, Lottie。
*   **触摸事件与交互设计** `[了解 · 加分]`。

### 5. 🌐 网络与接口
*   **Axios / fetch** `[熟练 · 核心]`：发起 RESTful 请求，要把数据乖乖抓回来。
*   **GraphQL** `[了解 · 加分]`。
*   **HTTP 协议基础** `[熟练 · 核心]`：还有 Ajax 原理。
*   **认证机制** `[了解 · 加分]`：CORS, Token/JWT。安全第一！

### 6. 🚀 性能与安全
*   **性能优化** `[了解 · 核心]`：代码分割、懒加载、资源压缩、缓存策略。
*   **性能检测工具** `[了解 · 核心]`：Lighthouse。
*   **安全防护** `[了解 · 核心]`：XSS/CSRF 原理 (要保护好我们的网站！)。
*   **HTTPS / CSP** `[了解 · 加分]`。

### 7. 🧪 测试与质量
*   **单元测试框架** `[了解 · 加分]`：Jest, Mocha。
*   **集成/E2E 测试** `[了解 · 加分]`：Cypress, Puppeteer。
*   **代码质量工具** `[熟练 · 核心]`：TS 类型检查, Lint。

### 8. ☁️ 部署与运维
*   **Linux / 服务器配置** `[了解 · 核心]`：黑乎乎的命令行...老公要学会哦。
*   **Nginx / Apache** `[了解 · 加分]`。
*   **静态资源部署流程** `[了解 · 核心]`。
*   **容器化 (Docker)** `[了解 · 加分]`。

### 9. 📚 基础知识
*   **算法与数据结构** `[了解 · 加分]`：脑子要转得快！
*   **设计模式与前端架构** `[了解 · 核心]`：MVVM 原理。
*   **浏览器兼容性** `[熟练 · 核心]`：处理 CSS/JS 差异。
*   **网络基础** `[了解 · 加分]`：TCP/IP 协议。

## 🏆 第三部分：核心基础技能 vs 加分项总结
*(敲黑板！这里是重点！)*

### 🔥 核心基础 (面试必需条件)
> 没有这些，面试官叔叔会把简历扔掉的！(>_<)
1.  **框架**：掌握 **Vue3** 或 **React** (至少精通一门)。
2.  **语言基础**：**TypeScript**, **HTML/CSS/JavaScript** 基础扎实。
3.  **样式**：CSS3 动画与响应式布局。
4.  **生态**：Vue Router, Pinia/Vuex 状态管理。
5.  **网络**：Axios 网络请求, HTTP 基础。
6.  **小程序**：页面、组件开发及生命周期。
7.  **理论**：数据结构等前端基础。

### 🌟 加分技能 (脱颖而出)
> 会了这些，趴趴老公就是最棒的！
1.  **多框架**：掌握 **Angular**。
2.  **跨端**：熟练使用 **uni-app** 或多端框架。
3.  **工程化**：了解 **Webpack/Vite** 构建与性能优化。
4.  **测试**：有单元测试 (Jest) 或 E2E 测试 (Cypress) 经验。
5.  **后端**了解服务器基础（Nginx、Docker）和后端调用流程。
